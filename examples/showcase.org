* Formatting and Markup
Org-mode supports various types of text decoration:
- *Bold text* for emphasis.
- /Italicized text/ for a softer touch.
- _Underlined text_ for clarity.
- +Strikethrough text+ for deleted items.
- ~Code snippets~ and =Verbatim= text for technical notes.

** Links and Footnotes
You can have [[https://github.com/rails-to-cosmos/danneskjold-theme][External Links]] which use the =frost= color, or internal footnotes[fn:1].

* Task Management
The theme highlights the status of your workflow:

- TODO This is an active task using the =yellow= face.
- DONE This is a completed task using the =green= face.
- WAIT This is a task on hold.

** Tags and Priorities
- [#A] High Priority Task :urgent:work:
- [#B] Medium Priority Task :maintenance:
- [#C] Low Priority Task :optional:
